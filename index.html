P5terms

A P5 sketch utilizing the P5.js speech library created by R. Luke DuBois and the SentiWordnet 3.0 database. The sketch takes the microphone input and analyzes the subjective emotional score of the sentence

View on GitHub Download .zip Download .tar.gz

p5Terms is a p5.js sketch developed by Zachary Witt at the University of Waterloo. The sketch utilizes the SentiWordnet 3.0 database to query inputted audio to create a visual representation of the subjective emotion related to the phrase.
